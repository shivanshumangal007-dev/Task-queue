# Task Queue Frontend

React + Vite dashboard for monitoring and interacting with the task queue backend.

## Scripts

- `npm run dev`: start Vite development server
- `npm run build`: create production build
- `npm run preview`: preview production build
- `npm run lint`: run ESLint

## Environment

Create a `.env` file in this folder (or copy from `.env.example`):

```
VITE_BACKEND_URL=http://localhost:3000
```

If `VITE_BACKEND_URL` is not set, the app defaults to `http://localhost:3000`.
